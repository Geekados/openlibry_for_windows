# openLibry *Windows-Installer*
*Installiert OpenLibry (Basis: https://github.com/jzakotnik/openlibry) inkl. NodeJS portable unter Windows*

## 🚀 Installation
1. Installer in der aktuellsten Version herunterladen

2. Installer ausführen 

3. openLibry starten (Desktop-Verknüpfung)<br>
   *Startet openLibry im Hintergrund, das Browserfenster öffnet sich automatisch*<br>
   
4. openLibry ist über den Browser mit dem Link http://localhost:3000 erreichbar<br>
   *Hinweis: das schwarze Fenster muss geöffnet bleiben, solange openLibry genutzt wird!*


## Features im Start-Script:
- Automatisches erstellen eines Backups der Datenbank undbenutzerspezifischen Daten bei jedem Start von openLibry
- Definierte Aufbewahrungszeiträume, automatisches Bereinigen der ältesten Backups
