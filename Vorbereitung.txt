Vorbereitung:
Nodejs portable herunterladen und in 
	C:\InnoInstaller\OpenLibry_Build\node 
entpacken 
Quelle: https://nodejs.org/en/download/archive/v22.22.1

openLibry herunterladen und in
	C:\InnoInstaller\OpenLibry_Build\app
entpacken

Install-Scripte in 
	C:\InnoInstaller\OpenLibry_Build\scripts
	-> Scripte: "install_helper.bat" und "start_lib.bat" im Git-Repo.

WICHTIG:
>> .env-datei erstellen und anpassen (Linux/Windows - Unterschiede bei den Pfaden beachten!)
>> die Ordner "database" und "images" erstellen (oder die .env anpassen)
>> next.config.js modifizieren: Zeile "output: 'standalone'," löschen

openLibry kompilieren:

:: 1. In den App-Ordner wechseln
cd /d C:\InnoInstaller\OpenLibry_Build\app

:: 2. Pfad zur portablen Node-Version setzen
set PATH=C:\InnoInstaller\OpenLibry_Build\node;%PATH%

:: 3. Sicherstellen, dass Prisma und Client da sind (wichtig für npx)
npm install prisma @prisma/client

:: 4. Restliche Abhängigkeiten (ohne Test-Tools wie Cypress)
npm install --omit=dev

:: 5. Prisma Client generieren (basiert auf deiner schema.prisma)
npx prisma generate

:: 6. Datenbank-Struktur abgleichen
npx prisma db push

:: 7. Den Next.js Produktions-Build erstellen
npm run build

Danach mit innoSetup (https://jrsoftware.org/isinfo.php) unter Verwendung der .iss-Datei kompilieren
